== Description ==

The file uploads addon adds an upload icon to comment boxes, which allows the user to upload a file with their comment.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

Check out the [Documentation](https://help.projecthuddle.com/article/122-file-uploads-setup/) for detailed information!
